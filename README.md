
# VOT - Video Organizer Tool

> A sleek, dark-themed YouTube video organizer that helps you structure your learning journey with hierarchical categories and privacy controls.

## Features

### Smart Organization
- **Hierarchical Categories** - Tree-like structures (Programming → Web Dev → Frontend → HTML)
- **Udemy-style Interface** - Clean sidebar with expandable categories
- **Unlimited Nesting** - Create as many subcategories as needed

### Privacy Control
- **Selective Locking** - Lock sensitive categories with access codes
- **Hierarchical Security** - Each subcategory can have independent locks
- **Temporary Access** - Unlocks expire after 1 month
- **Private Code Storage** - Secure unlock codes in separate files

### User Experience
- **Dark Theme** - Easy on eyes during long sessions
- **Mobile Responsive** - Works on all devices
- **Quick Video Access** - Play YouTube videos directly in modal
- **Progress Tracking** - Mark videos as completed
